# Repository locations

**Purpose**: Provide information about my setup for repository locations

## Where things are

- Work (infracost) related code repos: `~/repos/work` or `$HOME/repos/work`
- Personal code repos: `~/repos/personal` or `~/$HOME/repos/personal`

## Directives

- Default to always list/grep the work repos for surrounding context.
- Only list/grep personal repos if instructed to.
